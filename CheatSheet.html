Intro to HTML - Cheatsheet

HTML(Hypertext Markdown Language) is a coding language that is written in elements. An element is charazcterized by an opening and closing tags, and content within them.

Display Text
Tag Description
< p > Paragraph
< span > Short Text
< h1 > Headers. There are 6 levels of headers!
< br > Line breaks. Do not need a closing tag.

Lists
Tag Description
< ul > Unordered Lists. Will show a bulleted list.
< ol > Ordered Lists. Will show a numbered list.
< li > List items. Use for each item to be displayed.

Media
Tag Description
< img > Self closing tag. You must include an src attribute to source the image. You can also include an alt attribute to denote alternative text to improve SEO and accessibility.
< video > Include an src attribute to source the video. You should also include width, height and controls attributes to make it better. Add text within the tags, this will show up if the video isn't avaiable.
